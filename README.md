# Projeto 3.2: Jogo "Adivinhe o Número"

`Objetivo:` Desenvolver um jogo simples onde o computador "pensa" num número e o utilizador
tem de adivinhar qual é. 

`Requisitos:`
1. Armazene um número secreto numa variável (ex: numero_secreto = 7).
2. Use um ciclo while para permitir que o utilizador continue a adivinhar enquanto não
acertar.
3. Dentro do ciclo, peça um palpite ao utilizador com input() e converta-o para um número
inteiro.
4. Use uma estrutura if/elif/else para comparar o palpite com o número secreto e dar uma
dica: "Muito alto!", "Muito baixo!" ou "Correto!".
5. O jogo deve terminar quando o utilizador acertar, e uma mensagem de parabéns deve
ser exibida.
